### üöÄ Overview
**YT-Morphe** is a highly optimized, automated build system for YouTube patches. By leveraging the power of **MorpheApp**, it provides a premium, ad-free experience with enhanced features for both Root and Non-Root devices.

---

### üõ†Ô∏è Installation Modes

> [!IMPORTANT]
> Choose the version that best matches your device's state.

#### üèóÔ∏è **Root Version (Magisk/KernelSU)**
- **File:** `YT-Hybrid.zip`
- **Installation:** Flash directly via Magisk or KernelSU manager.
- **Benefits:** Seamless system integration, no extra apps required for login.

#### üì± **Non-Root Version**
- **File:** `YT.apk`
- **Prerequisite:** Requires [GmsCore](https://github.com/ReVanced/GmsCore) for account login.
- **Installation:** Sideload as a normal APK.

#### ‚ö° **Rooted APK Version**
- **File:** `YT-rooted.apk`
- **Usage:** Alternative APK version for rooted devices. Can be installed directly without Magisk module.
- **Note:** This version can be installed like a normal APK.
